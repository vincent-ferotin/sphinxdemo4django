Introduction¶

But¶

Le présent projet SphinxDemo sert à illustrer une utilisation possible de Sphinx pour la documentation d’un nouveau projet Django.

Prérequis¶

Ce projet suppose une connaissance préalable basique de Django (concepts, usages) et de ce à quoi sert Sphinx.
