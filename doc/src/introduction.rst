Introduction
=============

But
---

Le présent projet `SphinxDemo` sert à illustrer une utilisation possible
de `Sphinx`_ pour la documentation d'un nouveau projet `Django`_.


Prérequis
---------

Ce projet suppose une connaissance préalable basique de `Django`_
(concepts, usages) et de ce à quoi sert `Sphinx`_.
